# Personal Portfolio Website
This is a simple, responsive portfolio website that showcases my background, skills, experience, and contact information.

## Pages
- Home: Overview of who I am and what I do
- Resume: Professional experience and qualifications
- Contact: Email and LinkedIn details

## Features
- Responsive layout using Bootstrap
- Clean, simple design
- Downloadable resume (PDF)
- Easy contact options

## Tech Stack
- HTML
- CSS (Bootstrap)
- JavaScript / jQuery

## How to Run
Download or clone the repository, then open `index.html` in your browser.
Alternatively, view it live: https://dariojhoyte.github.io/site-dh-portfolio/index.html
